The statistic $\Delta_{AB}$ is scale-free: positive values favor target $A$, negative values favor target $B$, and zero is the knife-edge. ``Common support'' is the intersection of the one-step support sets under verified current-cell and age-based current matching; the union, $A$, and $B$ support columns report how much of the target mass remains on the stated support set. Panel B uses the point log-ratios in Panel A to decompose the shift into support exclusion and within-support rematching; it is computed from the underlying unrounded point statistics, so displayed entries need not match arithmetic on separately rounded numbers beyond the last shown decimal. This nearby PVEU-margin target-pair exercise uses the same financing rule and represented local margin as the main-text comparison.
