# Source notes for the main manuscript tables

The main tables in Section 8 are regenerated by `python/run_final_diagnostics.py`.

Maintained graph object: exhaustive current-cell tolerance graph with epsilon 0.005. It has 72,171 vertices, 249,501 edges, 3,737 components, and 181,067 independent cycles. Diagnostic five-nearest objects are used only in rows explicitly labeled as diagnostic.

The exact table source map is `python/outputs/MAIN_TABLE_SOURCE_MAP.csv` and is duplicated at the package root as `MAIN_TABLE_SOURCE_MAP.csv`.

Replication package posting address: https://github.com/sbuhai/current-cell-matching-project
